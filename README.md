# Huon's project folio

This is currently v2 of my folio. Check the branches for the v1 code.

# Cloning this project

### Clone repo

`git clone https://github.com/OMEN44/folio.git`

### Install dependencies

```sh
cd client
npm i
```

### Run client and server

#### Client

`npm run dev`

#### Server

`npm run server`
